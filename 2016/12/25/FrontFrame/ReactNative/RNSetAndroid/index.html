Microsoft C++ 环境

安装java JDk

从Java官网下载Java SE Development Kit 7 Downloads并安装。请注意选择x86还是x64版本。推荐将JDK的bin目录加入系统PATH环境变量。（安装JDK、JRE）

配置环境变量：

系统变量→新建 JAVA_HOME 变量

系统变量→寻找 Path 变量→编辑

系统变量→新建 CLASSPATH 变量→编辑

java -version 查看Java版本

安装Python 2

从官网下载并安装python 2.7.x（3.x版本不行）
